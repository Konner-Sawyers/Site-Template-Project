This project is an ongoing effort in which I am attempting to create a rendering library of sorts in order to assit with other projects. I enjoy creating simple simulations of things such as gravity sims and collision detectors however I find myself wanting a tailor made renderer to help kickstart my projects faster. The purpose of this program is to enable that desired faster development and to further improve my skills with graphical programming, mathematics and general C++ while using the SDL library as a base.

Currently the program supports:

  • The creation of objects with sprites and basic world space information

  • Keyboard input and event handling

  • 2D World Space to screen space along with camera and object translation

  • Proper scaling based on camera zoom

With plans to add:

  • Z-Axis creating 3D environment

  • Object scaling based on depth instead of set camera zoom
