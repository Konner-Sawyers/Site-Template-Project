A project done along side two others developing a Metabase dashboard to create a data warehouse for exoplanet information. Information was taken directly from NASA and the ESA Gaia archives and transformed into standarized information to be stored in the same database, which was then analyzed with a series of queries to discover trends in exoplanet discovery methods.
